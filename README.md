# SFHTMLPython

SFHTMLPython is a simple web application built with **Flask**, **Bootstrap**, and **Paramiko** that allows users to upload and download files through **SFTP**.  
It provides a clean HTML/CSS interface for file management and a Python backend to handle secure file transfers.
